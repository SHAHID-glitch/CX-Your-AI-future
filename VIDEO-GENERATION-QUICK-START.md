## Troubleshooting

### Q: Modal won't open
**A**: Check if JavaScript is enabled, refresh page

### Q: Video generation fails
**A**: 
- Check API connection
- Ensure HuggingFace API key is valid
- Try again in a few seconds
- Try a shorter description

### Q: Video won't play
**A**:
- Ensure browser supports HTML5 video
- Try a different browser
- Check if file is fully downloaded

### Q: Download not working
**A**:
- Check browser download settings
- Disable popup blockers
